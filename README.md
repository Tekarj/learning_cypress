<h1>Learning Cypress</h1>
</head>
<body>

> Tools: 
```
VSCode, Node.js
```
> Checking versions: 
```
npm -v / node -v
```
> Start communication between VSCode and Node.js: 
```
npm init
```
> Install cypress:
```
npm i cypress --save-dev
```
> Run/Open Cypress
```
npm run test
```
or
```
npx cypress open
```
